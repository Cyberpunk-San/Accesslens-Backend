# Security Policy

## Supported Versions

The following versions of AccessLens are currently being supported with security updates:

| Version | Supported |
| ------- | :-------: |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

We take the security of AccessLens seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, please report it privately by:

**Contacting the project owner immediately.**

### Our Commitment
- We will acknowledge receipt of your report within 48 hours.
- We will provide a timeline for fixing the issue.
- We will notify you once the vulnerability is resolved.

Thank you for helping keep AccessLens secure!
